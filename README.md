# **Tempestra**

**Tempestra** is a Fabric 1.20.1 visual overhaul for Minecraft thunderstorms and lightning. This repo unofficially ported it to Neoforge 1.21.1.

This mod replaces the normal vanilla lightning visual with thinner, more natural-looking lightning bolts that arc through the sky, glow, and can sometimes split before hitting the ground. **Tempestra** also adds stronger storm intensity, making thunderstorms feel more powerful and atmospheric without making them feel out of place in Minecraft.

**Tempestra** is meant to be a simple, purely visual improvement for Vanilla+, survival, fantasy, and atmospheric modpacks.
